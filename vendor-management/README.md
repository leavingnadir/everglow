# Vendor Management System

A dedicated vendor management system for the Everglow project. This module handles vendor registration, authentication, profile management, and service/package management.

## Features

- Vendor registration and authentication
- Vendor profile management
- Vendor authorization and roles
- Service/package management by vendors
- Vendor dashboard

## Structure

### Backend
- Java Spring Boot application
- RESTful API endpoints for vendor operations
- Integration with main everglow database

### Frontend  
- React-based vendor interface
- Vendor dashboard
- Registration and authentication flows
- Profile and service management pages

## Integration

This system is integrated with the main everglow backend and shares the same database and authentication mechanisms.
